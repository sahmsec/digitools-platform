# **DigiTools Platform**
<img width="100" height="100" src="./public/favicon.svg">

*React + Vite Application for Premium Digital Tools Marketplace*

🌐 Live: https://digitools-platform-sahmsec.netlify.app/

---

The **DigiTools Platform** is a modern web application built with **React, Vite, and Tailwind CSS** that allows users to explore, compare, and purchase premium digital tools.

### **✨ Key Features**
-  Product marketplace with detailed cards  
-  Cart system with real-time updates  
-  Pricing plans with highlighted tiers  
-  Smooth animations and responsive UI  

---

## **🛠️ Tech Stack**
| Category       | Technology                          |
|----------------|-------------------------------------|
| Frontend       | React                               |
| Build Tool     | Vite                                |
| Styling        | Tailwind CSS + DaisyUI              |
| State Mgmt     | React Hooks (`useState`, `use`)     |
| Icons          | Lucide React, React Icons                        |
